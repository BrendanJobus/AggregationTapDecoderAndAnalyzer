## About The Project

### Description
Decoder of Aggregation Taps from header and analysis of data from the Aggregation Taps.

## Getting Started

### Compile
* Compile the project
	```sh
	g++ main.cpp -lpcap -o test
	```

### Run
* Run the project
	```sh
	./test
	```
